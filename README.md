# Requirament
PHP 7.4 / 8.1
Composer PHP Version 2

# Instalasi
1. Clone project ini
2. masuk kedalam directory, jalankan
```
composer install
```
3. copy file .env.example menjadi .env
4. Masukkan konfigurasi koneksi ke s3 server di .env
5. jalan php artisan key:generate
6. jalan php artisan serve
